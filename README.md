# تطبيق أحلام كوفي - Ahlam Coffee

## نبذة عن المشروع

تطبيق أحلام كوفي هو تطبيق موبايل ل ordering القهوة والمشروبات، تم تطويره باستخدام Flutter مع هيكلة MVC وGetX. تم تصميم التطبيق لتلبية متطلبات مشروع جامعي ويتضمن جميع الميزات الأساسية لتطبيق التجارة الإلكترونية البسيطة.

## مميزات التطبيق

- **واجهة مستخدم ثنائية اللغة**: العربية والإنجليزية
- **نظام تسجيل دخول وحسابات**: تسجيل جديد وتسجيل دخول
- **قائمة المنتجات**: عرض منتجات القهوة والمشروبات
- **سلة طلبات ذكية**: إضافة وتعديل وحذف المنتجات
- **نظام إدارة الطلبات**: تتبع الطلبات والمحتوى
- **واجهة مستخدم سهلة الاستخدام**: تصميم أنيق وتجربة مستخدم ممتازة
- **هيكلة MVC مع GetX**: هيكلة معمارية نظيفة وفعالة

## الغرض من المشروع

- تطبيق مفتوح المصدر يظهر أفضل الممارسات في تطوير تطبيقات Flutter
- نموذج مرجعي لتطبيق التجارة الإلكترونية البسيطة
- توضيح استخدام هيكلة MVC مع مكتبة GetX
- مثال عملي لنظام إدارة الطلبات والمنتجات

## كيفية تنزيل وتثبيت المشروع من GitHub

### 1. استنساخ المشروع

```bash
git clone https://github.com/ahlamalsalami/CoofeeApp.git
```

### 2. الانتقال إلى مجلد المشروع

```bash
cd CoofeeApp
```

### 3. تثبيت التبعيات

```bash
flutter pub get
```
# 4. اضبط بياناتك محليا اذا كنت مرتبط بحساب اخر 
git config user.name "ahlamalsalami"
git config user.email "ahlamalsalami@gmail.com"

# 5. أضف كل الملفات
git add .

# 6. احفظ التغييرات
git commit -m "مشروع متجر القهوة"

# 7. أضف رابط المستودع
git remote add origin https://github.com/ahlamalsalami/CoofeeApp.git
او
git remote add origin https://ahlamalsalami@github.com/ahlamalsalami/CoofeeApp.git
# 8. الآن ادفع وسيطلب منك بيانات الدخول
git push -u origin main
## تشغيل التطبيق

### عبر الأوامر

لتشغيل التطبيق على جهازك:

```bash
flutter run
```

### عبر Visual Studio Code

1. فتح المشروع في VS Code
2. الضغط على `Ctrl+Shift+P`
3. كتابة "Flutter: Run" وتحديد الخيار
4. اختيار الجهاز الهدف (جهاز فعلي أو محاكي)

### عبر Android Studio

1. فتح المشروع في Android Studio
2. تحديد الجهاز الهدف من القائمة المنسدلة
3. النقر على زر "Run"

## هيكلة الملفات

```
lib/
├── models/          # نماذج البيانات
│   ├── auth.dart    # نموذج المصادقة
│   └── cart.dart    # نموذج السلة
├── views/           # مكونات الواجهة
│   ├── home_page.dart      # صفحة البداية
│   ├── auth_page.dart      # صفحة المصادقة
│   ├── products_page.dart  # صفحة المنتجات
│   ├── orders_page.dart    # صفحة الطلبات
│   └── settings_page.dart  # صفحة الإعدادات
├── controllers/     # وحدات التحكم
│   ├── auth_controller.dart
│   └── cart_controller.dart
├── bindings/        # ربط الاعتمادات
├── routes/          # توجيهات التطبيق
├── services/        # خدمات التطبيق
├── languages/       # ملفات الترجمة
└── utils/           # أدوات مساعدة
```

## مكونات النظام

### 1. نموذج (Model)
- **auth.dart**: يتعامل مع بيانات المستخدم وحالة تسجيل الدخول
- **cart.dart**: يدير محتويات سلة الطلبات

### 2. واجهة (View)
- **home_page.dart**: الصفحة الرئيسية
- **auth_page.dart**: صفحة تسجيل الدخول والاشتراك
- **products_page.dart**: عرض قائمة المنتجات
- **orders_page.dart**: إدارة سلة الطلبات
- **settings_page.dart**: إعدادات التطبيق

### 3. وحدة تحكم (Controller)
- **auth_controller.dart**: يدير عمليات المصادقة
- **cart_controller.dart**: يتحكم في وظائف السلة
